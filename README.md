# rava-seo

一个极简、中文优先的 VPN SEO 引流页项目。

目标：
- 面向中文浏览器 + 中文 IP 用户
- 用简洁清晰的落地页承接 SEO 流量
- 突出中国网络环境可用、3 天免费试用、4 台设备、不留日志
- 通过按钮把用户导向 Rava 的邀请链接

## 当前页面方向

首页重点卖点：
1. 自研协议，针对中国网络环境穿透力更强
2. 3 天免费试用
3. 一个账号支持 4 台设备
4. 不留日志，保护隐私
5. 支持 Android / iPhone / Windows / Mac

## 本地开发

```bash
npm install
npm run dev
```

## 生产构建

```bash
npm run build
npm run start
```

## 当前路由

- `/`：中文引流首页
- `/go`：跳转到主邀请链接
- `/robots.txt`
- `/sitemap.xml`
